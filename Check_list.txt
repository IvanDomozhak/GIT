Название чек-листа:
Проверка функциональности и интерфейса поисковой строки в Яндекс.Браузере

1. Функциональность
Шаг 1: Ввод текста в поисковую строку
Действие: Ввести текст "Яндекс Браузер" в поисковую строку.
Ожидаемый результат: Введенный текст отображается полностью, без искажений.
Шаг 2: Выполнение поиска
Действие: Нажать на кнопку "Найти" или клавишу Enter.
Ожидаемый результат: Открывается страница с результатами поиска, которые соответствуют запросу.
Шаг 3: Автозаполнение подсказками
Действие: Начать ввод слова "погода".
Ожидаемый результат: Отображается выпадающий список релевантных подсказок.
Шаг 4: Голосовой ввод
Действие: Нажать на иконку микрофона и произнести "новости".
Ожидаемый результат: Поисковая строка распознает текст и выполняет поиск.
Шаг 5: Очистка строки
Действие: Нажать на крестик для удаления текста.
Ожидаемый результат: Поисковая строка становится пустой.
2. Интерфейс и отображение
Шаг 6: Проверка внешнего вида строки
Действие: Открыть Яндекс.Браузер и проверить отображение строки.
Ожидаемый результат: Строка поиска видна полностью, текст читаем, элементы не перекрываются.
Шаг 7: Адаптивность
Действие: Уменьшить и увеличить размер окна браузера.
Ожидаемый результат: Элементы строки поиска остаются на месте, пропорции сохраняются.
Шаг 8: Подсказки в поисковой строке
Действие: Ввести текст "фильм".
Ожидаемый результат: Отображается выпадающий список подсказок, не перекрывающий другие элементы.
3. Совместимость
Шаг 11: Проверка на разных платформах
Действие: Открыть Яндекс.Браузер на Windows, macOS, Linux.
Ожидаемый результат: Все функции поисковой строки работают одинаково на всех платформах.
Шаг 12: Проверка на разных устройствах
Действие: Проверить работу строки на ПК, ноутбуке, планшете, смартфоне.
Ожидаемый результат: Строка корректно работает на всех устройствах.
